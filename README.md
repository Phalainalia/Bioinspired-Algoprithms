# Bioinspired Algorithms

> This series of Colab Notebook projects explores and plays with the uses of some of the most famous bioinspired algorithms.

---

## Overview

This project uses mainly vanilla python to make examples and exercises that solve problems such as the TSP using bioinspired algorithms.

Various google Colabs are added, their main use was experimentation and exploring.



## Models

- Bees Algorithms
- Genetical Algorithms
- Genetical Algorithm (Using Special Heuristics)
